# Автономні веб-ескізи блогу CMS

Цей каталог містить повністю автономні веб-ескізи, створені на основі файлів CMS з папки cms2. Усі файли оптимізовані та готові для використання без зовнішніх залежностей.

## Структура файлів

### Основні сторінки

1. **index.html** - Головна сторінка блогу
   - Показує привітальне повідомлення після створення блогу
   - Містить навігацію та посилання для створення нового запису

2. **settings.html** - Сторінка налаштувань блогу
   - Форми для зміни назви блогу, автора, опису
   - Налаштування доступу та теми оформлення
   - Працює у автономному режимі

3. **new_post.html** - Сторінка створення нового запису
   - Форма для введення заголовку та тексту запису
   - Підтримка завантаження зображень
   - Можливість збереження як чернетки

4. **post.html** - Сторінка перегляду запису
   - Повний текст запису з форматуванням
   - Система коментарів
   - Відображення міток та метаданих

## Особливості реалізації

### Вбудовані стилі
- Всі CSS стилі вбудовані безпосередньо в HTML файли
- Підтримка темної та світлої теми через CSS змінні
- Адаптивний дизайн для різних пристроїв

### Автономна функціональність
- Видалені всі зовнішні посилання на CSS та JavaScript файли
- Мінімальний JavaScript для базової інтерактивності
- Заміна серверної логіки на заглушки з логуванням

### Візуальні елементи
- SVG іконки вбудовані безпосередньо в HTML
- Заповнювачі для аватарок користувачів
- Готові форми та кнопки

## Як використовувати

1. **Перегляд у браузері**: Відкрийте будь-який HTML файл у веб-браузері
2. **Навігація**: Використовуйте посилання в навігаційній панелі для переходу між сторінками
3. **Інтерактивність**: Форми та кнопки працюють у демо-режимі з логуванням у консоль

## Структура CSS

Стилі засновані на фреймворку E2 (Егея) та включають:

- **CSS змінні** для кольорової схеми
- **Flexbox та Grid** для розміщення елементів
- **Медіа-запити** для адаптивності
- **Анімації та переходи** для плавності
- **Типографіка** з підтримкою різних шрифтів

## Приклади використання

### Запуск у браузері
```bash
# Відкрити файл у браузері за замовчуванням
open index.html

# Або скопіювати шлях до файлу та відкрити в браузері
```

### Локальний сервер (опціонально)
```bash
# Використання Python
python -m http.server 8000

# Використання Node.js
npx serve .

# Використання PHP
php -S localhost:8000
```
